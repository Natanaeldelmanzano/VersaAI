🛠️ Stack Tecnológico

⚛️

Frontend Moderno

React 18 con TypeScript, Material-UI para componentes elegantes, Redux Toolkit para gestión de estado y Vite para desarrollo rápido.

🐍

Backend Robusto

Django REST Framework con autenticación JWT, Celery para tareas asíncronas y WebSockets para comunicación en tiempo real.

🗄️

Base de Datos

PostgreSQL para datos relacionales, Redis para caché y sesiones, ChromaDB para vectores y búsqueda semántica.

🤖

IA Avanzada

Sistema RAG con embeddings, integración Groq API, modelos LLM múltiples y procesamiento de lenguaje natural.

🐳

DevOps

Docker Compose para desarrollo, CI/CD automatizado, monitoreo con Prometheus y deployment escalable.

🔒

Seguridad

JWT tokens, hashing PBKDF2, CORS configurado, rate limiting, encriptación de datos sensibles y 2FA.
